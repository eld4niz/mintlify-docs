# Installation Guide

Complete installation guide for Neuronav SLAM SDK on different platforms.

## Choose Your Platform

<Tabs>
  <Tab title="Ubuntu 20.04/22.04">
    ## Prerequisites

    ```bash
    # Update system
    sudo apt update && sudo apt upgrade -y

    # Install essential tools
    sudo apt install -y curl git python3-pip build-essential cmake
    ```

    ## Install ROS2 Humble

    ```bash
    # Add ROS2 repository
    sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key \
      -o /usr/share/keyrings/ros-archive-keyring.gpg

    echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
      http://packages.ros.org/ros2/ubuntu $(lsb_release -cs) main" \
      | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

    # Install ROS2
    sudo apt update
    sudo apt install ros-humble-desktop ros-dev-tools -y

    # Setup environment
    echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
    source ~/.bashrc
    ```

    ## Install SLAM Dependencies

    ```bash
    # RTAB-Map
    sudo apt install ros-humble-rtabmap-ros -y

    # Additional packages
    sudo apt install ros-humble-imu-filter-madgwick \
                     ros-humble-foxglove-bridge \
                     ros-humble-image-transport-plugins -y
    ```

    ## Install Camera Drivers

    ### For Intel RealSense
    ```bash
    # Add Intel repository
    sudo mkdir -p /etc/apt/keyrings
    curl -sSf https://librealsense.intel.com/Debian/librealsense.pgp | \
      sudo tee /etc/apt/keyrings/librealsense.pgp > /dev/null

    echo "deb [signed-by=/etc/apt/keyrings/librealsense.pgp] \
      https://librealsense.intel.com/Debian/apt-repo $(lsb_release -cs) main" | \
      sudo tee /etc/apt/sources.list.d/librealsense.list

    # Install
    sudo apt update
    sudo apt install librealsense2-dkms librealsense2-utils \
                     ros-humble-realsense2-camera -y
    ```

    ### For OAK-D Pro
    ```bash
    # Install DepthAI
    pip3 install depthai opencv-python
    sudo apt install ros-humble-depthai-ros -y

    # Setup udev rules
    echo 'SUBSYSTEM=="usb", ATTRS{idVendor}=="03e7", MODE="0666"' | \
      sudo tee /etc/udev/rules.d/80-depthai.rules
    sudo udevadm control --reload-rules && sudo udevadm trigger
    ```

    ## Install Neuronav SDK

    ```bash
    # Clone repository
    git clone https://github.com/neuronav/neuronav-slam-sdk.git
    cd neuronav-slam-sdk

    # Install Python package
    pip3 install -e .

    # Verify installation
    python3 -c "from neuronav import RealSenseSensor; print('✅ SDK installed!')"
    ```

  <Tab title="Windows 11 (WSL2)">
    ## Enable WSL2

    ```powershell
    # In PowerShell as Administrator
    wsl --install -d Ubuntu-22.04

    # Restart computer when prompted
    ```

    ## Configure WSL2

    ```bash
    # Inside WSL2 terminal
    sudo apt update && sudo apt upgrade -y

    # Enable systemd (required for ROS2)
    echo -e "[boot]\nsystemd=true" | sudo tee /etc/wsl.conf

    # Exit and restart WSL
    exit
    # In PowerShell: wsl --shutdown
    # Then reopen WSL
    ```

    ## USB Camera Access

    ```powershell
    # Install USBIPD (in Windows PowerShell)
    winget install usbipd

    # List USB devices
    usbipd list

    # Attach camera to WSL (replace 2-4 with your device)
    usbipd attach --wsl --busid 2-4
    ```

    ## Continue with Ubuntu Steps

    Follow the Ubuntu installation steps above inside WSL2.
  </Tab>

  <Tab title="Docker">
    ## Install Docker

    ```bash
    # Install Docker Engine
    curl -fsSL https://get.docker.com -o get-docker.sh
    sudo sh get-docker.sh

    # Add user to docker group
    sudo usermod -aG docker $USER
    newgrp docker

    # Verify Docker
    docker run hello-world
    ```

    ## Build Neuronav Image

    ```bash
    # Clone repository
    git clone https://github.com/neuronav/neuronav-slam-sdk.git
    cd neuronav-slam-sdk

    # Build image
    ./docker_build.sh
    # Or manually:
    docker build -t neuronav-slam .
    ```

    ## Run Container

    ```bash
    # With device access
    ./docker_run.sh

    # Or manually:
    docker run -it --rm \
      --privileged \
      --network host \
      --volume /dev:/dev \
      --volume $(pwd):/workspace \
      --env DISPLAY=$DISPLAY \
      --volume /tmp/.X11-unix:/tmp/.X11-unix \
      neuronav-slam
    ```

    ## Docker Compose

    ```yaml
    # docker-compose.yml
    version: '3.8'
    services:
      neuronav:
        image: neuronav-slam
        privileged: true
        network_mode: host
        volumes:
          - /dev:/dev
          - ./data:/data
        environment:
          - ROS_DOMAIN_ID=1
        command: python3 /workspace/examples/basic_slam.py
    ```

    ```bash
    # Run with compose
    docker-compose up
    ```

## Verify Installation

### Test Camera Detection

```bash
# For RealSense
realsense-viewer

# For OAK-D
python3 -c "import depthai; print('OAK-D ready!' if depthai.Device.getAllAvailableDevices() else 'No device')"
```

### Test ROS2

```bash
# Check ROS2
ros2 topic list

# Should show system topics
```

### Test Neuronav SDK

```python
# test_install.py
from neuronav import RealSenseSensor, run_slam, RTABMapSLAM

try:
    sensor = RealSenseSensor()
    print("✅ Sensor initialized")

    slam = RTABMapSLAM()
    print("✅ SLAM initialized")

    print("✅ Installation successful!")
except Exception as e:
    print(f"❌ Error: {e}")
```

## Common Installation Issues

### Permission Denied

```bash
# Add user to required groups
sudo usermod -a -G dialout $USER
sudo usermod -a -G video $USER
# Log out and back in
```

### ROS2 Not Found

```bash
# Source ROS2 setup
source /opt/ros/humble/setup.bash
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
```

### Camera Not Detected

```bash
# Check USB connection
lsusb | grep -E "Intel|Luxonis"

# Reset USB
sudo modprobe -r uvcvideo
sudo modprobe uvcvideo
```
